# DETAILLOR — Next.js Rebuild

A modern, scalable rebuild of the DETAILLOR website using Next.js, TypeScript, and Tailwind CSS.

## Project setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Architecture

- `src/app` — Next.js App Router entry points
- `src/components` — reusable layout, UI, and section components
- `src/data` — structured mock data for products and categories
- `src/lib` — shared TypeScript interfaces and helper types
- `src/styles` — Tailwind global styles and custom overrides

## Deployment

This project is configured for deployment on Vercel.
